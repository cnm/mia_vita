FILE *.brd -> Circuit Board layout
FILE *.sch -> Circuit Schematic
FILE *.cam -> Cam processor config to generate Gerber files

The pdf's exports sould be done in a way so they face each other.

Gerber Files:
*.GTL -> Top Layer Copper
*.GBL -> Bottom Layer Copper
*.GTO -> Top Layer Silkscreen
*.GTP -> Top Layer Paste
*.GBO -> Bottom Layer Silkscreen
*.GTS -> Top Layer Soldermask
*.GBS -> Bottom Layer Soldermask
miavita.txt -> Drill Information File
